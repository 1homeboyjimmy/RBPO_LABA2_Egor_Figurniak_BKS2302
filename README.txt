Тема - Склад

Сущности
Item: товар
Warehouse: склад
Stock: остаток
Movement: перемещение
Supplier: поставщик

Операции
Перемещение товара между складами
Резервирование
Отчёт по остаткам
Уведомления о низком уровне (Показывает товары, у которых количество ниже определённого порога (< 10))
Проверка порога (Проверяет, не опустился ли остаток ниже заданного порога для конкретного товара)
